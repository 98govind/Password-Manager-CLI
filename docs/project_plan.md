# Password Manager CLI - Project Plan

## Goal
Build a secure command-line password manager to store and manage passwords.

## Core Features
- Master password login
- Add credential
- View credential
- Delete credential
- Generate strong password

## Security Features
- Encrypt stored passwords
- Keep master password protected
- Copy password to clipboard

## Future Features
- Search by service name
- Password strength checker
- Session lock
- Backup and restore

## Tech Stack
- Python
- CLI
- Encryption [library]
